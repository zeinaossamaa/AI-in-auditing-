# AI-in-auditing-
Artificial Intelligence in Auditing

This project presents an AI-supported auditing framework developed as part of a bachelor thesis at the German University in Cairo (GUC). The research explores how Artificial Intelligence technologies such as Machine Learning (ML), Natural Language Processing (NLP), and Robotic Process Automation (RPA) can improve traditional auditing processes through enhanced risk assessment, anomaly detection, automation, and decision-making support while maintaining transparency, explainability, and professional auditor judgment.

The repository includes the thesis documentation, BPMN diagram, use case diagram, prototype wireframe, and supporting project files. The BPMN diagram should be opened using  Camunda Modeler￼, while the use case diagram should be opened using  Draw.io￼. The prototype and wireframes were designed using  Figma.

The proposed framework supports the main auditing phases including audit planning, evidence gathering, audit execution, and reporting. The system integrates AI technologies to support risk scoring, anomaly detection, NLP-generated summaries, automated reporting, and real-time audit analytics through a balanced human-AI collaboration approach.
